# VibeCoding Workflow Rules

这些规则是 VibeCoding 工作流的质量门禁。

## 规则列表

| 规则文件 | 说明 | 优先级 |
|---------|------|-------|
| vibecoding-requirements.md | 需求必须包含边界条件和业务规则 | HIGH |
| vibecoding-tdd.md | 严格 TDD（Red-Green-Refactor） | HIGH |
| vibecoding-testing.md | 综合验收测试（功能/安全/性能/异常/兼容/数据库） | HIGH |
| vibecoding-human-review.md | 安全/支付代码必须人工审核（AI扫描+CI门禁） | HIGH |
| vibecoding-commit.md | 提交和 PR 约定格式与合并策略 | HIGH |
| vibecoding-memory.md | .vibecoding/ 长期记忆文件维护规则 | HIGH |
| vibecoding-documentation.md | 新功能必须生成文档，含位置约定 | MEDIUM |
