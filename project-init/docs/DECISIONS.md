# 架构决策记录（ADR）

> **目的**：记录「为什么这样设计」，防止 AI 在后续对话中反复质疑已有决策。
> 
> **写入时机**：做了架构选型后立即追加一条。**只追加，不覆盖历史。**

---

## 使用说明

每条决策包含：

| 字段 | 说明 |
|------|------|
| **ID** | ADR-001 递增 |
| **日期** | YYYY-MM-DD |
| **决策** | 一句话描述做了什么选择 |
| **背景** | 为什么需要做这个决策 |
| **方案对比** | 考虑过的方案及优劣 |
| **最终选择** | 选了哪个 + 核心理由 |
| **后果** | 带来的好处和需要承担的代价 |
